## Project Setup Commands

```bash
# 1. Create Next.js project (if not exists)
npx create-next-app@14 . --typescript --tailwind --eslint --app --src-dir=false --import-alias="@/*" --use-npm

# 2. Install dependencies
npm install sanity next-sanity @sanity/image-url @sanity/vision resend zod lucide-react clsx tailwind-merge

# 3. Install dev dependencies
npm install -D prettier prettier-plugin-tailwindcss
```

### Post-Setup Config Changes

- Update `tailwind.config.ts` with custom theme (primary colors, fonts)
- Update `next.config.js` with image domains (cdn.sanity.io)
- Create `sanity.config.ts` at project root
- Create `sanity/` directory for schemas and lib
