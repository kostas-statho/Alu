## Recommendations

### Before Deployment

1. **Create Sanity project** — Run `npx sanity init` or create via sanity.io dashboard, then populate `.env.local`:
   ```
   NEXT_PUBLIC_SANITY_PROJECT_ID=your-project-id
   NEXT_PUBLIC_SANITY_DATASET=production
   SANITY_API_TOKEN=your-token
   ```

2. **Configure Resend** — Sign up at resend.com, get API key, add to `.env.local`:
   ```
   RESEND_API_KEY=re_...
   RESEND_FROM_EMAIL=noreply@yourdomain.com
   ```

3. **Set site URL** — Update `NEXT_PUBLIC_SITE_URL` in `.env.local` and `robots.txt`

4. **Configure Plausible** — Add analytics script (after domain setup)
